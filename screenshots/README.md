# Screenshots

This folder contains screenshots of the HishabKhata application.

## How to Add Screenshots

To complete the documentation, please add the following screenshots:

### 1. Light Theme (`light-theme.png`)
- Open `index.html` in a browser
- Ensure the app is in light mode (default)
- Add some sample data to all three tables (Cash, Dues, Loans)
- Take a full-page screenshot showing:
  - All three tables with data
  - The summary cards at the bottom
  - The theme toggle button in the top-left

- Open browser developer tools (F12)
- Switch to mobile device view (responsive mode)
- Select a phone size (e.g., iPhone 12, Galaxy S20)
- Take a screenshot showing:
  - The responsive layout on mobile
  - How the tables stack on smaller screens

## Screenshot Specifications

- **Format**: PNG (preferred) or JPG
- **Resolution**: Full resolution of your screen
- **File Size**: Optimize if larger than 500KB
- **Aspect Ratio**: Natural viewport ratio

## Tools for Screenshots

### Windows
- **Snipping Tool**: Win + Shift + S
- **Game Bar**: Win + G
- **Browser Extensions**: Fireshot, Awesome Screenshot

### Browser DevTools
- Chrome/Edge: F12 ‚Üí Device Toolbar (Ctrl + Shift + M)
